Anotações

Para que o campo sejá obrigatório, usar required(exemplo neste HTML)

Para númerto minimo ou máximo, usar minlength ou maxlength

Pode usar a opção placeholder para incluir coisas dentro das caixas

A função autocomplete(on ou off) pode ser usada para bloquear ou permitir que apareça as opções de auto completar dos navegadores

Usando dentro de cada input, é possível dizer para que serve aquela caixa(nome, e-mail, organização, telefone...) e assim deixar o navegador mais acertivo na hora de oferecer opções para o usuário.
