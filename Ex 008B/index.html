Código fonte

Quando queremos escrever um código, algumas fontes são ruins, pois cada caracter vai ter um tamanho diferente, por isso podemos usar o código "code".

Por exemplo o comando de JS document.getElementbyid('teste') pode ser ruim de ler. Este porém tem a tag code aplicada: getElementbyid('teste')
