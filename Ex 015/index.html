Importante

+ de 1 "style"

Pode-se criar quantos "style.css" quiser, bastando linkar as páginas que desejar.

Por exemplo, você deseja que algumas páginas do site tenham caracteres especificos além do CSS geral, basta criar essa página de CSS e linkar

Por exemplo:
link rel="stylesheet" href="style.css"
link rel="stylesheet" href="estiloEspecifico.css"
