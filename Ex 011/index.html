Imagens dinâmicas

Para que apareça imagens com tamanhos diferentes apra dispositivos com tamanhos de telas diferentes, é preciso usar o código a seguir, entre a função %lt picture>

<source media="(max-width:000px)" srcset="imagens/foto-p.png"

Aonde o "imagens/foto-p.png" é o nome da imagem e o 00px é o tamanho que você quer a quebra. Importante destacar que tem que fazer um desses para cada imagem e cada ponto de quebra

Reproduzindo áudio

Vamos aprender a reproduzir áudio

A maneira mais fácil é com a tag <audio>, usar também o parametro <controls, para aparecer os controles>

Porém nem sempre o navegador vai entender, por isso é interessante colocar mais de um tipo de arquivo(mp3,wav,ogg..)

É interessante colocar sempre o parametro "preload", que tem 3 opções: None, Metadata e autoplay.

Neste caso, o ideal é colocar none ou metadata, pois se for autoplay, o navegador só vai entender que o site carregou todo depois de carregar todo o arquivo.
